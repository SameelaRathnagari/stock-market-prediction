📈 Real-Time Stock Market Analysis and Price Prediction System
Overview

The Real-Time Stock Market Analysis and Price Prediction System is a web-based application developed using Python and Streamlit. The application provides real-time stock market insights, company information, technical analysis indicators, interactive visualizations, and machine learning-based stock price prediction.

The system integrates with the Yahoo Finance API to fetch historical and live stock market data and helps users analyze stock performance through various financial indicators and predictive analytics.

Installation

Clone the repository:

git clone https://github.com/SameelaRathnagari/stock-market-prediction.git

Install the required dependencies:

pip install -r requirements.txt

Run the application:

python -m streamlit run app.py

Open the application in your browser:

http://localhost:8501
